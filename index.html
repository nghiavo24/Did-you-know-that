How to play

Before starting the game, you can enter your name in the box. Click the Start button to start the trivia. The question will be shown with a list of possible answer. If you select the correct answer, the answer will turn green and if the answer is wrong, it will turn red. There is a score on top to keep track of your progress. Try to get as many correct answers as possible. You can restart the game anytime during the game by clicking on the Reset button.
